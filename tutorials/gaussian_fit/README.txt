In this folder you can find the tutorial to run a gaussian fit in three different applications.
To run the Bayesian Inference test provided in this folder follow the following instructions:
1 - Open the GaussianFit.cpp file and use the compilation line provided on top (see Installation Guide for more information)
2 - Execute the program via terminal in the same tutorials folder, making sure to match the dataset with the proper prior hyper parameters file. 
A set of output ASCII files containing the results will be stored in the same folder.
